═══════════════════════════════════════════════════════════════════
                    KEY CONCEPTS
═══════════════════════════════════════════════════════════════════

Backend:
├─ Express.js server running on port 3000
├─ Receives form submissions via POST
├─ Validates data
└─ Saves to complaints.json

Frontend:
├─ index.html is the main website
├─ Form sends data to backend API
├─ Displays success/error messages
└─ Updates in real-time

API:
├─ POST /api/complaints → Submit new complaint
├─ GET /api/complaints → Get all complaints
├─ GET /api/complaints/:id → Get one complaint
└─ GET /api/health → Check server status

Data:
├─ Stored in complaints.json
├─ JSON format with fields
├─ Unique ID per complaint
└─ Timestamps for tracking


═══════════════════════════════════════════════════════════════════
                    COMMAND REFERENCE
═══════════════════════════════════════════════════════════════════

Start the server:
  npm start

Stop the server:
  Ctrl + C (in terminal)

Install dependencies:
  npm install

View complaints data:
  cat complaints.json

Check if server is online:
  curl http://localhost:3000/api/health

Get all complaints:
  curl http://localhost:3000/api/complaints

Make a complaint (curl):
  curl -X POST http://localhost:3000/api/complaints \
    -H "Content-Type: application/json" \
    -d '{"fullName":"Name","wardNumber":"Ward 1","issue":"Issue"}'

Open website in browser:
  open index.html

Check backend status page:
  open status.html
