🎯 FILTERING BEHAVIOR
═══════════════════════════════════════════════════════════════

[ALL] button active:
├─ Shows all complaints
├─ Both pending (yellow) and resolved (green)
├─ Default view

[PENDING] button active:
├─ Shows only pending complaints
├─ Only yellow status badges
├─ Hide resolved items

[RESOLVED] button active:
├─ Shows only resolved complaints
├─ Only green status badges
├─ Hide pending items
