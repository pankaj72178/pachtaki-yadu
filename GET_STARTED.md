## 🆘 TROUBLESHOOTING

### "Cannot connect to server"
```
Step 1: Check server is running (npm start)
Step 2: Check both devices on same WiFi
Step 3: Try http://localhost:3000 on Mac first
Step 4: Check if port 3000 is open
```

### "Page loads but no data"
```
Step 1: Refresh page (Ctrl+R)
Step 2: Hard refresh (Ctrl+Shift+R)
Step 3: Clear browser cache
Step 4: Check browser console (F12)
```

### "Connection refused"
```
Step 1: Wait 5 seconds after npm start
Step 2: Check IP in browser matches server output
Step 3: Try http://127.0.0.1:3000 first
Step 4: Restart server: Ctrl+C then npm start
```

### "Only works on Mac"
```
Step 1: Check firewall settings
Step 2: Use IP from server output
Step 3: Make sure same WiFi network
Step 4: Try different browser on other device
```
