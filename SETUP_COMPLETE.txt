📞 TROUBLESHOOTING QUICK FIXES

  Problem: "Cannot connect"
  Solution:
  1. Check: npm start (server running?)
  2. Check: Same WiFi (both devices?)
  3. Check: Correct IP (from server output)
  4. Fix: Restart server

  Problem: "Data not syncing"
  Solution:
  1. Refresh page
  2. Hard refresh (Ctrl+Shift+R)
  3. Clear cache
  4. Check server logs

  Problem: "API error"
  Solution:
  1. Check server running
  2. Check correct IP:port
  3. Check network connection
  4. Restart if needed

  More Help:
  → See QUICK_START_NETWORK.md
  → See NETWORK_SETUP.md
